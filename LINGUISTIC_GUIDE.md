# LinkLedger: Linguistic Guide

All UI text and labels in this application must follow these specific linguistic constraints.

## 1. Forbidden Starting Sequences
Do not use any words that start with:
- **F**
- **E**
- **M** (Exception: **MONTHLY**)
- **Ov**
- **Re**
- **Dr**

## 2. Recommended Swaps

| Forbidden | Use Instead |
| :--- | :--- |
| **F**eatures | **Capabilities** |
| **E**xperience | **Journey** |
| **M**anagement | **Control** |
| **Re**port | **Audit** |
| **Re**newal | **Cycle** |
| **Re**move | **Delete** / **Trash** |
| **Dr**op | **Trash** / **Shed** |
| **Ov**erall | **Total** |

---
*Maintained to ensure a unique and symmetrical logic.*
